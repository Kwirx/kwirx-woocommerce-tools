A comprehensive toolkit for WooCommerce store management, featuring tools for image duplicate removal, product dimension updates, and more.

== Description ==

Kwirx WooCommerce Tools is a plugin designed to enhance your WooCommerce store management experience. It provides a set of tools to streamline your workflow and improve your product data management.

Key features:
* Image Duplicate Remover: Automatically detect and remove duplicate product images, saving storage space and improving site performance.
* Dimension Updater: Easily update product dimensions in bulk, ensuring accurate shipping calculations and improved customer experience.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/kwirx-woocommerce-tools` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the WooCommerce -> Kwirx Tools screen to access the plugin's features.
